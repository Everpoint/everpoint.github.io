snappingMethods

snappingMethods: SnappingMethod[]
  • Defined in controls/snapping/SnappingProviderBase.ts:31

List of snapping methods to be used for finding the snapping point. The methods are applied in the order of this list. So the snapping point from first method to return a value other then null will be used for snapping.
