getLayers

  • getLayers(recurse?: boolean, excludeInactive?: boolean): Layer[]
    • Defined in LayerGroup.ts:202

    Returns the flat list of layers in the group. Child LayerGroups are not included.

    Parameters

    • Default value recurse: boolean = false

      if set to true, the contents of child LayerGroups will be added to the list in the corresponding order. E.g. if the group contents are layer1, group1, layer2, then the resulting list will be the result of concatenation of [layer1], group1.getLayers(true), [layer2].

    • Default value excludeInactive: boolean = false

      if set to true, layers with isDisplayed=false and all their children will not be included.

    Returns Layer[]

insertLayer

  • insertLayer(layer: GroupItem, index: number): void
    • Defined in LayerGroup.ts:164

    Inserts the layer to the given position. If the layer is already in the group, moves the layer so that new index of the layer equals the specified index. If the index is negative, layer is added to the n-th position from the end of the list. If the index is larger than number of the layers in the group, layer will be added to the end of the list.

    fires

    ContentsChangeEvent

    throws

    If the given layer cannot be added to the group (e.g. if the result of reordering creates recursive nesting).

    Parameters

    • layer: GroupItem

      layer to insert.

    • index: number

      integer position of the layer after insertion.

    Returns void
