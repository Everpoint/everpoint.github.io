Interface ISnappingProvider

Snapping providers are used by feature creation/editing controls to snap to some relevant points.

getSnappingPoint

  • getSnappingPoint(point: Coordinates, activeContour?: Contour, activeIndex?: number, isPolygon?: boolean): Coordinates | null
    • Defined in controls/snapping/ISnappingProvider.ts:18

    This method is called by a control to find a snapping point close to the given point. If a relevant snapping point not found, the method must return null.

    Parameters

    • point: Coordinates

      this is usually where the mouse cursor is at the current moment. So snapping point should be in vicinity of this point.

    • Optional activeContour: Contour

      in case of editing a polygon or polyline, this argument will contain the contour (set of points) that is being edited at the moment.

    • Optional activeIndex: number

      in case of editing a polygon or polyline, this argument will be an index of the point in 'contour' that is being edited at the moment.

    • Optional isPolygon: boolean

      specifies if the feature being edited should be considered enclosed.

    Returns Coordinates | null
