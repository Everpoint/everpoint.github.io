Class ServiceOperation

Base class for all service operations. Service operation is a method of the server, available for invoking by the client. A child must implement exactly one method with attribute ServiceOperationMethodAttribute and return type IMessageResponse or Task<IMessageResponse> (or Task of one of its implementations).
