Style

Style is a specification for a layer about how the features should be drawn. To do that, the layer calls SelectStyle(QueryFeature)method for its style for each feature to draw, and then uses selected style's Symbol to draw the feature. The style must be compiled with Compile(AttributesDefinition) method before SelectStyle(QueryFeature) can be called.
